FEATURE: Grid Cells

In Foundation, grids are a great way to split up your work!
In order to create a comprehensive grid, you must have a divider for the grid-x grid-padding-x, the type of grid cell, and a primary callout.

This is a twelve cell section in a grid-x. In this case, the class for the divider would be a large-12 cell.

Six cell: the class here is large-6 medium-6 cell.

Another six cell :)

Four cell: the class here is large-4 medium-4 small-4 cell.

FEATURE: Buttons

Buttons can be really helpful tools when you want to complete more traditional actions. Foundation has many ways to use certain styles styles that you can fit your needs with.

When class="tiny button", we get... a tiny button
When class="small button", we get... a slightly bigger button
When class="large button", we get... a BIG BUTTON

FEATURE: Cards

Cards are a popular and flexible UI component in Foundation!
- class="card" initiates the creation of a card
- class="card-divider" divides it into two sections
- class="card-section" allows you to write content in the card
